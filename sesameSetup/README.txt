---------------------------------------------------------------------------
                        SESAMESETUP
---------------------------------------------------------------------------

This folder is the set up folder for the sesame repository, used to connect
with the 'Fundamental Relationship configuration tool'.
It contains the following folders:

1)applications: 
  ------------
  This folder contains necessary software and applications for the 
  sesame repository, in order to be able to run it standalone.
  
2)bin:
  ---
  This folder contains a series of .bat files, used to start and stop the 
  sesame server.
  
3)ontotext_owlim:
  --------------
  This folder contains the files regarding the rules of the repository, 
  created using OWLIM. Also here we keep the used ontologies schemata.
  
4)sesame_data:
  ------------
  Here the repositories are created and their data are kept
